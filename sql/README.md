# SQL Directory

This folder contains all SQL scripts used in the Healthcare Claims Analytics project.

## 📂 Folder Structure

- **ddl/** → Table creation scripts (schema)
- **etl/** → Data transformation and aggregation logic
- **kpi/** → Key metric calculations (e.g., biosimilar_penetration, biosimilar_new_start rate)


## 📊 Key Outputs

- `client_annual_summary` → built in `/etl`
- High cost claimant calculation → `/kpi`


## 🧠 Notes

- All queries are written in standard SQL
- Scripts are modular and reusable
- Naming conventions reflect business purpose
