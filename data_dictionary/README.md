# Data Dictionary

This folder contains table-level data dictionary files for the Healthcare Claims Analytics project.

## Files

- `claims.txt` — prescription claims table
- `clients.txt` — client or plan sponsor dimension
- `members.txt` — member demographic dimension
- `drug_master.txt` — drug reference table
- `industries.txt` — standardized industry lookup
- `client_annual_summary.txt` — annual client KPI summary table
- `member_specialty_drug.txt` — specialty drugs claimed by members
- `specialty_assignment.txt` — specialty drug tags for members with specialty drug claims
- 

## Purpose

These files document:
- table definitions
- column descriptions
- data types
- key relationships
- table grain
